# Release Note 模板

```markdown
# {产品名} - V{版本号}
本次release note 包含{起始版本}-{版本号}的所有更新内容

## 发布时间
{YYYY-MM-DD}

## 主要功能与优化

### 功能新增
- [module] {条目}

### 功能变更
- [module] {条目}

## 问题修复
- [module] {条目}

## 注意事项
- {条目}
```

## 填写约束

- `{产品名}` 示例：`Middleware`
- `{版本号}` 示例：`1.2.0`（不带前缀 `v`）
- `{起始版本}` 为本次汇总起点版本，例如 `1.1.0`
- 每条变更必须带 `[module]` 前缀；无法归类时使用 `[misc]`
- 某章节无内容时，写 `-`
- 仅输出 Markdown 正文，不输出解释

## 示例

```markdown
# Middleware - V1.2.0
本次release note 包含1.1.0-1.2.0的所有更新内容

## 发布时间
2026-02-24

## 主要功能与优化

### 功能新增
- [deploy] 中间件支持整包离线部署（除ORIN镜像外）
- [upperlimb] 新增多动作调用接口 多动作文件调用

### 功能变更
- [upperlimb] WA1下示教模式 禁止升降示教
- [sensor] Sensor镜像支持Ubuntu22.04系统

## 问题修复
- [robot] 修复/zj_humanoid/robot/basic_info话题只返回固定值的问题

## 注意事项
- 由于1.2.0的导航感知版本不再维护，故该版本不包含导航感知，若后续使用感知导航请使用1.3.0版本。
```
